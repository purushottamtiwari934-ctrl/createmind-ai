# CreateMind AI — Project Specification

## Project Type

Premium production-ready website for CreateMind AI, an AI-powered creative content production studio.

## Business Purpose

CreateMind AI helps creators, startups, brands, businesses, and agencies produce high-impact creative content using artificial intelligence and modern creative workflows.

## Core Services

- AI Video Production
- Short-Form Content for Reels and Shorts
- AI Product Visuals
- Static Visual Assets
- Ad Creatives
- Brand Commercials
- AI Campaign Creative Production
- Creative Direction
- High-Volume Content Production

## Website Goal

The website must present CreateMind AI as a premium AI creative production studio and convert visitors into project inquiries.

Primary actions:

- Start a Project
- Get a Quote
- Explore Our Work
- View Packages
- Request Consultation

## Design Direction

Use a premium cinematic dark theme.

The design should include strong modern typography, subtle gradients, elegant glow effects, smooth scrolling, refined micro-interactions, and a creative technology studio aesthetic.

Avoid generic SaaS dashboard design, excessive animation, visual clutter, and cheap AI-style graphics.

The website must be fully responsive and mobile optimized.

## Footer

Include CreateMind AI branding, short brand description, navigation links, services, social links, contact area, and copyright notice.

## Technical Requirements

Use a modern full-stack architecture supported by the build environment.

Use reusable components.

Keep services, pricing, and portfolio content in centralized structured data.

Use semantic HTML.

Add SEO metadata and Open Graph foundations.

Optimize performance and mobile responsiveness.

Use environment variables for secrets.

Never expose credentials.

Keep the architecture clean, scalable, and maintainable.

## Scope Restrictions

Do not add authentication.

Do not add a user dashboard.

Do not add payment systems.

Do not add an unnecessary CMS.

Do not remove required website sections.

Prioritize the premium website experience, inquiry flow, SEO foundation, performance, and maintainable architecture.
